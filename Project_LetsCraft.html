Let's Craft

Role: Game Designer
Genre: 3D Sandbox
Company: Ten Square Games
Platform: Mobile / Facebook Instant

For Let's Craft, an experimental block-based mobile building game playable with one hand, I've created the game map (full workflow)
and all in-game 3D voxelart models. The map had to conform to a very tight technical specification and the project had a short deadline.
I had to come up with biomes and models that would be consistent with each other, while creating as much variety as possible since we
focused heavily on player-generated content. Additionaly I've handled documentation maintenance and game balance.

- Full map creation workflow, from concept to the finished product
- Created all in-game 3D models using MagicaVoxel
- Designed game balance & player progression
